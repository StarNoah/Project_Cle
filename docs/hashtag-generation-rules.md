# 암장 해시태그 자동 생성 규칙

구현: `lib/gym-sync.ts` > `generateHashtags()`

## 필터링

- `입구`로 끝나는 항목 제외 (CSV의 나머지는 모두 포함)

## 생성 규칙

공백 제거한 이름을 기준으로, 아래 규칙을 순서대로 적용한다.

| 규칙 | 조건 | 처리 |
|------|------|------|
| 전체 이름 | 항상 | 공백 제거 후 추가 |
| ~점 삭제 | "점"으로 끝날 때 | 마지막 "점" 제거 |
| 시설 접미사 제거 | 센터/센타/짐/장으로 끝날 때 | 해당 접미사 제거 |
| 브랜드 추출 (공백 있음) | 이름에 공백이 있을 때 | 첫 번째 단어 |
| 브랜드 추출 (공백 없음) | 이름에 공백이 없을 때 | 클라이밍센터/센타/짐/장/클라이밍/볼더링 접미사 제거 |
| 짧은 브랜드 제외 | 브랜드가 1글자일 때 | 추가하지 않음 |

## 예시

| 이름 | 결과 |
|------|------|
| 더클라임 양재점 | 더클라임양재점, 더클라임양재, 더클라임 |
| 오프더월클라이밍 | 오프더월클라이밍, 오프더월 |
| 에이스클라이밍센터 | 에이스클라이밍센터, 에이스클라이밍, 에이스 |
| 인천베스트클라이밍센타 | 인천베스트클라이밍센타, 인천베스트클라이밍, 인천베스트 |
| 인클라이밍센터 | 인클라이밍센터, 인클라이밍 (브랜드 "인" 제외) |
| 포고클라임 | 포고클라임 |
